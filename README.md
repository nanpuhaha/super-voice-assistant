# Super Voice Assistant

macOS voice assistant with global hotkeys.

## Getting Started

1. **Build**

   ```bash
   chmod +x build_app.sh
   ./build_app.sh
   ```

2. **Install**
   Move the generated `SuperVoiceAssistant.app` to your **Applications** folder.

3. **Run & Authorize**
   Open the app and grant **Microphone** and **Accessibility** permissions.
   _(Accessibility is required for global hotkeys and auto-pasting)._

4. **Select Model**
   In Settings, download **whisper v3 turbo** (Recommended).

5. **Use**
   Press `Command + Option + Z` to start/stop listening.
